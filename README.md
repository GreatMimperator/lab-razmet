# Лабораторная работа №1: Сбор неструктурированных данных

**Тема:** Виды транспорта (10 объектов).
**Цель:** Собрать текстовые описания, изображения и аудиозаписи для последующей разметки и построения графа знаний.

### Состав датасета
В папку `data` включены следующие типы транспорта:
Вертолет, Пассажирский поезд, Мотоцикл, Самолет, Трамвай, Подводная лодка, Трактор, Пожарный автомобиль, Снегоход, Теплоход.

Для каждого объекта собраны:
1. Текст (.md) — краткое описание из Википедии (объем ~1 стр).
2. Изображение (.jpg/.png) — заглавное фото из статьи.
3. Аудио (.mp3/.ogg) — характерные звуки работы или сигналов.

### Структура проекта
* `data/` — итоговый датасет (папки audio, image, text).
* `replacements/` — папка для ручных правок. Файлы отсюда имеют приоритет и копируются в data автоматически.
* `transports.json` — входной список названий для парсинга.
* `parser.py` — скрипт для автоматизации сбора.

### Инструкция по запуску
Для работы требуются библиотеки `wikipedia-api` и `requests`.

1. Положить ручные аудиофайлы в `replacements/audio/`.
2. Запустить скрипт:
   `python parser.py` — режим обновления (докачивает недостающее).
   `python parser.py --clear` — полная очистка папки data и повторный сбор.

### Источники
* Текст и фото: [Wikipedia](https://ru.wikipedia.org/) (автоматический сбор).
* Аудио: [Pixabay](https://pixabay.com/sound-effects/) (ручной сбор).
